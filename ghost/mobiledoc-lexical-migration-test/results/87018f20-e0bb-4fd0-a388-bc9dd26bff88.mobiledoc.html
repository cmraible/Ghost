Overview

There weren’t a lot of changes for the 1981 Corvette from the previous year.

There weren’t a lot of changes for the 1981 Corvette from the previous year. The only engine available was a new reworking of the previous years’ 350ci V8. However, the new Computer Command Control electronically controlled the emissions and fuel system. It also featured an auxiliary electric cooling fan, which allowed for a smaller engine driven fan and reduced noise.
