Vivaldi is the web browser of choice here at BestTechie. Vivaldi is fast, secure, easy to use, and works great. It's also based on Chromium, the browser developed by Google and used to build Google Chrome which gives Vivaldi a lot of great features right out of the box.

As you would expect from a modern web browser, Vivaldi supports the latest web standards so every website you visit should appear correctly. Plus, the Vivaldi team is constantly developing Vivaldi and releases updates for it frequently which are automatically installed for you in the background so you can always be assured you’re running the latest and most secure version.

Key Features

  • Built-in ad and tracking blockers to help ensure your privacy as you browse the web. No need for third-party ad/tracking blocking extensions.
  • A feature called Speed Dials which allows you to quickly access your favorite websites from the Start Page.
  • Easily import your bookmarks and other content into Vivaldi from other browsers (including Google Chrome, Safari, and Firefox).
  • Sync your Vivaldi install with your Vivaldi account so you have the same tabs, bookmarks, and other content on all your devices.
  • Private mode allows you to browse the web without being tracked or leaving any browser history.
  • Because Vivaldi is based on Chromium you also have access to the entire library of Google Chrome themes and browser extensions to extend functionality.
  • No Google tracking. Because Vivaldi is an independent company, they have removed all of Google's tracking functionality in their browser.
  • For a full list of Vivaldi's features (and there are many), check out their website.

Currently Vivaldi is available on the desktop for macOS, Windows, and Linux. There's also an Android app for mobile. Unfortunately, there's not yet an iOS version of Vivaldi – but I'm hopeful we'll see one soon.
