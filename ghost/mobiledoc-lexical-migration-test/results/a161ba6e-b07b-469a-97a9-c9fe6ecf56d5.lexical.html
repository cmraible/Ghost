Slight changes were made for the 1959 Corvette; like the loss of unneeded chrome and hood louvers, which cleaned up the look. Rear-tailing rods were added and contributed to a softer ride and better handling. All of this added to a sportier image and reflected its true competitive spirit: the Corvette was racking up victories on the racetrack.
